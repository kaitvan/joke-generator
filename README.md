# Joke Generator

## Description
The goal of this project was to create a joke generator using an API.

### Feature List
The user can:
- View a randomly generated joke
- Click a button to show the answer
- Click a button to show a new joke

## Tech Used
* SASS
* HTML
* JS with ES6 modules
* jQuery
* Webpack configuration
